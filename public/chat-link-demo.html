Information about the Chat Link Implementation

How to Implement

To add a chat link to your website, include the following script with the mode=link parameter:

<script src="/api/embed-script?mode=link"></script>

You can customize the link text and styling:

<script src="/api/embed-script?mode=link&text=Ask%20about%20grants&title=Grant%20Assistant"></script>

Place an element with the data-ai-chat-link attribute wherever you want the link to appear:

<span data-ai-chat-link></span>
