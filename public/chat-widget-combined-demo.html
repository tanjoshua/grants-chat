Chat Assistant Integration Demo

This demo shows two different ways to integrate the chat assistant into your website.

Option 1: Floating Button

The floating button appears in the bottom-right corner of your website. Visitors can click it to open the chat in a popup.

Implementation

Add this script to your website:

<script src="/api/embed-script"></script>

That's it! The button will automatically appear in the bottom-right corner.

When to Use

  • You want the chat to be available throughout the entire site
  • You prefer a non-intrusive, always-accessible option
  • You want a consistent UI element across all pages

Result

Look at the bottom-right corner of this page - you'll see a chat button there!

Option 2: Chat Links

Chat links can be placed anywhere in your content. When clicked, they open the chat in a modal dialog.

Implementation

Add this script to your website:

<script src="/api/embed-script?mode=link&text=Chat with us"></script>

Then add this HTML wherever you want a chat link to appear:

<span data-ai-chat-link></span>

Implementation Details

Floating Button Implementation

The floating button widget creates a fixed-position button in the corner of your website. When clicked, it opens an iframe containing the chat interface.

Basic Implementation

<!-- Add this to your website's HTML -->
<script src="/api/embed-script"></script>

How It Works

  1. The script creates a floating button in the bottom-right corner
  2. When clicked, it opens an iframe with the chat interface
  3. The chat interface communicates with the parent window using postMessage
  4. The button can be clicked again to hide the chat

Link Widget Implementation

The link widget allows you to place chat triggers anywhere in your content. When clicked, they open a modal dialog containing the chat interface.

Basic Implementation

<!-- Add this to your website's HTML -->
<script src="/api/embed-script?mode=link"></script>

<!-- Add this wherever you want a chat link to appear -->
<span data-ai-chat-link></span>

Customizing Link Text

<!-- Custom text in the markup -->
<span data-ai-chat-link>Ask about grants</span>

<!-- Or set default text in the script -->
<script src="/api/embed-script?mode=link&text=Ask about grants"></script>

Styling as a Button

<!-- Add your own classes -->
<span data-ai-chat-link class="btn primary-btn">Chat Now</span>

Customization Options

Button Widget Parameters

// No customization parameters currently available

Link Widget Parameters

<script src="/api/embed-script?mode=link
  &text=Chat with us         // Default link text
  &title=Grant Assistant     // Dialog title
  &class=my-link-class"      // Additional CSS classes
></script>

Advanced Styling

You can add your own CSS to override the default styles of the widget components:

/* Button widget styling */
#ai-chat-widget-button {
  background-color: purple;
  width: 60px;
  height: 60px;
}

/* Link widget styling */
.ai-chat-link {
  color: #ff0000;
  font-weight: bold;
}

.ai-chat-dialog {
  border: 2px solid gold;
}
