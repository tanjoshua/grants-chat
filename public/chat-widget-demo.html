The Enterprise Development Grant supports projects that help Singapore companies grow and transform. This grant supports projects under three pillars: Core Capabilities, Innovation and Productivity, and Market Access.

Grant Amount: Up to 70% of qualifying costs

Eligibility Criteria

To be eligible for most government grants, businesses must:

  • Be registered and operating in Singapore
  • Have at least 30% local shareholding
  • Be in a financially viable position to start and complete the project

Not sure if you qualify? Check your eligibility with our assistant.

Implementation Details

Floating Button Implementation

The floating button widget creates a fixed-position button in the corner of your website. When clicked, it opens an iframe containing the chat interface.

Basic Implementation

<!-- Add this to your website's HTML -->
<script src="/api/embed-script"></script>

How It Works

  1. The script creates a floating button in the bottom-right corner
  2. When clicked, it opens an iframe with the chat interface
  3. The chat interface communicates with the parent window using postMessage
  4. The button can be clicked again to hide the chat

Link Widget Implementation

The link widget allows you to place chat triggers anywhere in your content. When clicked, they open a modal dialog containing the chat interface.

Basic Implementation

<!-- Add this to your website's HTML -->
<script src="/api/embed-script?mode=link"></script>

<!-- Add this wherever you want a chat link to appear -->
<span data-ai-chat-link></span>

Customizing Link Text

<!-- Custom text in the markup -->
<span data-ai-chat-link>Ask about grants</span>

<!-- Or set default text in the script -->
<script src="/api/embed-script?mode=link&text=Ask about grants"></script>

Styling as a Button

<!-- Add your own classes -->
<span data-ai-chat-link class="btn primary-btn">Chat Now</span>

Combined Widget Implementation

The combined widget provides both a floating button and in-content chat links with a single script. This is the recommended approach for most websites.

Basic Implementation

<!-- Add this single script to get both button and links -->
<script src="/api/embed-script?enableLinks=true"></script>

<!-- Add this wherever you want chat links to appear -->
<span data-ai-chat-link></span>

Available Parameters

<script src="/api/embed-script?enableLinks=true
  &text=Chat with us         // Default text for links
  &class=my-link-class"      // Additional CSS classes for links
></script>

Benefits of Combined Approach

  • Single script to manage for easier maintenance
  • Consistent user experience between button and links
  • Reduced page load with only one script
  • Both interfaces share the same chat backend

Advanced Styling

The same styling options apply to both interfaces:

/* Button widget styling */
#ai-chat-widget-button {
  background-color: purple;
  width: 60px;
  height: 60px;
}

/* Link widget styling */
.ai-chat-link {
  color: #ff0000;
  font-weight: bold;
}

.ai-chat-dialog {
  border: 2px solid gold;
}
