AI Chat Widget Implementation

This page demonstrates how to embed the AI Chat Widget on any website. You can see the chat widget in action by clicking the chat icon in the bottom-right corner.

Note: This is just a demonstration. The real chat widget would be integrated with your AI backend and customized to your needs.

How to Embed the Chat Widget

To add the chat widget to your own website, simply include the following script tag at the end of your HTML body:

<script src="https://your-domain.com/api/embed-script"></script>

Replace your-domain.com with your actual domain where the Next.js application is hosted.

Customization

The chat widget can be customized to match your website's branding and style. You can modify the colors, sizes, and other properties by editing the widget's source code.

Features

  • Responsive design that works on desktop and mobile
  • Real-time AI-powered chat using your custom model
  • Markdown support for rich formatting
  • Minimalist UI that can be customized to match your brand
